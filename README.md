# Frontend Web Development 1

**Bomjan, Gyalpozhing College of Information Technology**

This repository is mainly created for **daily homework and practice tasks** in **Frontend Web Development**.  
It documents the learning journey of building web pages using **HTML**, **CSS**, and **JavaScript (DOM)**.

---

## Purpose of This Repository

- Practice writing clean and semantic **HTML**
- Learn styling and layout techniques using **CSS**
- Understand how to interact with the browser using **JavaScript**
- Perform daily homework and class assignments
- Track development progress over time

---

## What This Repository Includes

### HTML

- Page structure and elements
- Lists, tables, images, forms
- Semantic tags and document organization

### CSS

- Selectors and styling rules
- Box model, positioning, flexbox
- Colors, fonts, responsive layouts

### JavaScript (DOM)

- Selecting and modifying elements
- Handling events (click, submit, input)
- Creating dynamic and interactive UI elements

---

## Learning Path

This repository follows a structured learning progression from beginner to advanced topics:

### Week 1-2: HTML Fundamentals
1.  [basic-elements](classwork/basic-elements/) - HTML structure, headings, links, images
2.  [lists-and-text-formatting](classwork/basic-elements/lists-and-text-formatting/) - Lists, text formatting, semantic elements

### Week 3-4: CSS Basics
3.  [css-link](classwork/css-link/) - Linking CSS to HTML
4.  [colors](classwork/colors/) - Color formats and properties
5.  [selectors-in-css](classwork/selectors-in-css/) - CSS selectors and specificity
6.  [fonts-and-typography](classwork/fonts-and-typography/) - Custom fonts and typography

### Week 5-6: CSS Layouts
7.  [grids](classwork/grids/) - CSS Grid and Bootstrap grid system
8.  [transformations](classwork/transformations/) - CSS transforms and transitions
9.  [loading](classwork/loading/) - CSS animations and keyframes
10. [bootstrap](classwork/bootstrap/) - Bootstrap framework

### Week 7-8: JavaScript & DOM
11. [dom-manipulation/shopping-list](classwork/dom-manipulation/shopping-list/) - Array manipulation and event delegation
12. [dom-manipulation/login-signup](classwork/dom-manipulation/login-signup/) - Local storage and authentication
13. [dom-manipulation/validation-classwork](classwork/dom-manipulation/validation-classwork/) - Form validation
14. [dom-manipulation/regex-validation](classwork/dom-manipulation/regex-validation/) - Regular expressions

### Week 9-10: Advanced Topics
15. [jquery-basics](classwork/jquery-basics/) - jQuery library fundamentals
16. [crud-app](classwork/crud-app/) - Full CRUD application

## Project Structure

This repository is organized into classwork and assignments:

### Classwork
-   **basic-elements**: Fundamental HTML elements (headings, links, images, paragraphs).
    -   **lists-and-text-formatting**: HTML lists and text formatting elements.
-   **bootstrap**: Introduction to the Bootstrap framework.
-   **colors**: Working with colors in CSS (RGB, HEX, HSL, named colors).
-   **crud-app**: A basic Create, Read, Update, Delete application.
-   **css-link**: Linking CSS to HTML (external, internal, inline styles).
-   **dom-manipulation**: Manipulating the DOM with JavaScript.
    -   **shopping-list**: A simple shopping list application.
    -   **login-signup**: Login and Signup forms with local storage.
    -   **validation-classwork**: Form validation examples.
    -   **regex-validation**: Regular expression patterns and validation.
-   **fonts-and-typography**: Using custom fonts and typography best practices.
-   **grids**: CSS Grid layouts and Bootstrap grid system.
-   **jquery-basics**: Introduction to jQuery with interactive examples.
-   **loading**: CSS loading animations with keyframes.
-   **selectors-in-css**: CSS selectors (class, id, element, attribute, pseudo-classes).
-   **transformations**: CSS transforms, transitions, and clip-path.

### Assignments
-   **assignment-1**: HTML navigation and images
-   **assignment-2**: Personal portfolio project
-   **lab-exercise-1**: Lab exercises
