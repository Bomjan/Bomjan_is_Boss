## Repository Overview

This repository documents my 14-week journey learning Frontend Web Development, covering:
- **HTML fundamentals** - Structure and semantics
- **CSS fundamentals & layouts** - Styling and responsive design
- **DOM manipulation** - Interactive web applications
- **Hands-on projects** - Practical application of concepts

**Note**: JavaScript fundamentals (variables, functions, loops, arrays, objects) are covered in the separate [**Fundamentals of Programming**](https://github.com/Bomjan/FoP) module.
