Schema Documentation

Reference guide for MongoDB Collections

Users Collection
+ _id ObjectId Unique identifier
+ firstName String
+ lastName String
+ dob Date
+ gender Char
+ dzongkhag String
+ gewog String
+ contact Object
+ role Enum doctor | patient
+ specialization String OPTIONAL
Medicines Collection
+ _id ObjectId Unique identifier
+ name String Name of the medicine
Records Collection
+ _id ObjectId
+ recordedAt Date
+ patient Object { id, name }
+ doctor Object { id, name, specialization }
+ medicine Array [{ id, name, duration }]
Appointments Collection
+ _id ObjectId
+ date Date
+ time Time
+ status Enum pending | confirmed | cancelled
+ title String
+ doctor Object { id, name }
+ patient Object { id, name }
